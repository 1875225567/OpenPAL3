# 仙三模拟器

（刚刚开工的）《仙剑奇侠传三》开源实现。

## 声明

- 仙三模拟器并不附带任何《仙剑奇侠传三》的游戏数据。您必须拥有《仙剑奇侠传三》的正版拷贝才可以正常运行仙三模拟器。
- 仙三模拟器并非软星公司或大宇集团的官方作品。
